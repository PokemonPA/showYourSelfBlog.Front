{
    code: " ",
    msg:  " ",

## 登陆

- 登陆

```json
url: "/api/login/login"
type:"POST"

{
    phone:" ",
    password:" "               #需要md5加密
}

return:
{
    code: 1001/1003
}
```

- 注册

  - 获取验证码

  ```json
  url: "/api/getCheckNum"
  type: "POST"
  
  {
    phone:" "
  }

  return:
  {
      code: 1001/1007
  }
  ```
  
  - 注册
  
  ```json
  url: "/api/login/register"
  type: "POST"
  
  {
      phone:" ",
      username:" ",
      password:" ",				#需要md5加密
      checkNum:" "				#验证码
  }
  
  return:
  {
  	code: 1001/1008
  }
  ```
- 忘记密码

  - 获取验证码

  ```json
  url: "/api/getCheckNum"
  type: "POST"
  
  {
      phone:" "
  }
  
  return:
  {
      code: 1001/1007
  }
  ```

  - 修改密码

  ```json
  url: "/api/login/forget"
  type: "POST"
  {
      phone: " ",
      checkNum: " "			#验证码
  }
  
  return:{
      code:
  }
  ```
